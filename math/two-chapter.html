Q.1 : নীচের কোন শর্ত মতে
p(x) = \(ax^2 + bx + c , a \neq{0} \) বহুপদ রাশির লেখ দুটির উপরের দিক খোলা অধিবৃত্ত অথবা নিচের দিক খোলা অধিবৃত্ত হয়

(A) a = 0, b = 0
(B) \(a\gt{0} ,a\lt{0} \)
(C) \(a\gt{0} \ ,b\lt{0} \)
(D) \(b\gt{0} ,a\lt{0} \)
Explanation
Correct Ans:(b)
(2) যদি y = f(x) বহুপদ রাশির লেখটি x- অক্ষক দুটা বিন্দুতে ছেদ করে, তবে f(x) এর শূণ্যে র সংখ্যা হবে –

(B) 2
(C) 3
(D) 4

(B) \(\frac{2}{3}\)
(C) -8
(D) 6

(D) \(\frac{3}{4}\)

(b) \(x^2 +x -6 \)

(c) \(MNa^2 = (M + N)^2 \ ac\)
(d) \(MNb^2 = (M + N)^2 \ ac\)

Ans: (d)

(b) 7

(a) p(x) এর মাত্রা \(\lt \ \)g(x) এর মাত্রা

Ans: (d)

(20) যদি p(x) = g(x) × q(x) + r(x) , এখানে r(x) = 0 অথবা _____

(a) r(x) এর মাত্রা \( \gt \ \) g(x) এর মাত্রা
(b) r(x) এর মাত্রা \( \leqslant \ \) g(x) এর মাত্রা
(c) r(x) এর মাত্রা \( = \ \) g(x) এর মাত্রা
(d) r(x) এর মাত্রা \( \lt \ \) g(x) এর মাত্রা
Explanation
Ans: (d)
(21) ত্রিঘাত বহুপদ রাশি \(ax^3 + bx^2 + cx + d \) এর একটি শূন্য 0 হলে বাকি দুটি শূন্যের গুণফল হবে

(a) \({-}\frac{c}{a}\)
(b) \(\frac{c}{a}\)
(c) 0
(d) \({-}\frac{b}{a}\)
Explanation
Ans: (b)
সমাধান:
দেওয়া আছে, একটি মূল 0
\(\therefore p(x) =ax^3 + bx^2 + cx + d = 0\)
\(\Rightarrow p(0) =0 + 0 + 0 + d = 0 \)
\(\Rightarrow d = 0 \)
সুতরাং, সমীকরণ হবে
\(x(ax^2 + bx + c)\)
অন্য দুটি মূল হবে \(ax^2 + bx + c \) এর
\(\therefore\) দ্বিঘাত সমীকরণের মূল দুটির গুণফল \(\frac{c}{a}\)
(22) n মাত্র বিশিষ্ট বহুপদ রাশির শূন্যের সংখ্যা হল
