Voice Change ‚Äì Full Rules

Simple Present
ActivePassive
He writes a letter.A letter is written by him.

Rule: S + V1 + O ‚Üí O + is/am/are + V3 + (by + S)

Present Continuous
She is reading a book.A book is being read by her.

Rule: S + is/am/are + V1+ing ‚Üí O+ is/am/are + being + V3

Simple Past
They played cricket.Cricket was played by them.

Rule: V2 ‚Üí was/were + V3

Present Perfect
I have done my homework.My homework has been done by me.

Rule: has/have + V3 ‚Üí has/have + been + V3

Modal Verbs
You can open the door.The door can be opened by you.

Rule: modal + V1 ‚Üí modal + be + V3

Imperative Sentences
Close the door.Let the door be closed.
Do not waste time.Let time not be wasted.

Rule: Let + object + be + V3

Interrogative (Yes/No)
Did she sing a song?Was a song sung by her?
Does he write letters?Are letters written by him?

Rule: Do/Does/Did + S + V1 + O ‚Üí Is/Are/Was/Were + O + V3 + (by + S)?

Interrogative (WH-type)
Who wrote this book?By whom was this book written?
When did he complete the work?When was the work completed by him?

Rule: WH-word + V ‚Üí WH-word + be verb + V3 + (by + S)?

Exclamatory Sentences
What a beautiful painting she made!What a beautiful painting was made by her!
How sweetly he sings!How sweetly the song is sung by him!

Note: Keep the exclamatory expression intact, just change the voice of the main clause.
